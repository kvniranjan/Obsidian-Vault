## What It Does
Ploi Roadmap is a self-hostable, open source product roadmapping and customer feedback management tool built with Laravel (PHP). It provides a public-facing portal where stakeholders and end users can submit feature requests, report bugs, and upvote existing items — giving product teams and business analysts a structured, transparent way to capture and prioritize the voice of the customer without relying on expensive SaaS tools like Canny or ProductBoard.

The platform supports a lightweight role system (administrator, employee, and user) so BAs can control who proposes items, who comments, and who votes. Each roadmap item moves through configurable workflow states (e.g., Planned, In Progress, Shipped, Declined), making it straightforward to communicate delivery progress to stakeholders in real time. Auto-generated slugs and Open Graph images mean the roadmap can be shared publicly or embedded in stakeholder communications with minimal effort.

A standout feature for technical BA teams is the GitHub integration, which links roadmap items directly to GitHub issues or milestones, creating a lightweight traceability chain from customer request to engineering delivery. An `/ai` endpoint exposes each item as machine-readable JSON, making it simple to feed roadmap data into Claude or other LLMs for automated reporting, stakeholder summaries, or backlog prioritization.

## Key Features
- Public roadmap portal for stakeholder, customer, and team feedback submission
- Upvoting system to quantify and surface the highest-demand features
- Role-based access control (administrator, employee, user)

- Self-hostable via standard Laravel or Docker deployment

## Installation
```bash
# Clone the repository
git clone https://github.com/ploi/roadmap.git
cd roadmap

# Install PHP dependencies
composer install

# Install JS dependencies and build frontend assets
npm install && npm run build

# Configure environment variables
cp .env.example .env
php artisan key:generate

# Run database migrations and seed initial data
php artisan migrate --seed

# (Optional) Configure GitHub integration in .env:
# GITHUB_TOKEN=your_token
# GITHUB_REPO=org/repo

# Start the development server
php artisan serve
```
