# TRL Upgrade

## Overview
TRL (Transaction Reporting Logic) upgrade for Brazil and Argentina contracts. Loans targeted for Aug-Sep, Deposits for Oct-Nov. Involves null value population for TRL20 (Evergreen maturity accounts), frequency code mapping fixes, and loan product mapping gap analysis. Argentina has 533 contracts needing maturity type corrections. Pre-certification required for 5 countries.
