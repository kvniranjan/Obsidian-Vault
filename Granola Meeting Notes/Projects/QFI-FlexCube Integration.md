# QFI-FlexCube Integration

## Overview
Replacing the current Evox/Ebooks production system with QFI for fixed income products (buy, sell, repos, reverse repos) in Brazil. The project is split into two phases — Phase 1 targets May production rollout, Phase 2 targets November completion. Integration uses a JSON Kafka model specific to Brazil's database structure.
