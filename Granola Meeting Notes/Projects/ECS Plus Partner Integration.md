# ECS Plus Partner Integration

## Overview
Onboarding ECS Plus (also known as JTF) as a new partner for Brazil. Involves deploying EOD/SOD notification services (enhanced for JSON via Kafka) and a new transaction posting service instance. Includes CIP integration for transaction response handling, GO code configuration (replacing MIS codes), cash account setup with account class decisions, and JTF onboarding configuration with user role mapping.
