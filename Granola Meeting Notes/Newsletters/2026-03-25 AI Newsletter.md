# AI Newsletter - 2026-03-25

## Top Stories

- **[Santander + Mastercard Complete Europe's First Live AI Agent Payment](https://fintechmagazine.com/news/santander-and-mastercard-complete-first-ai-payment)** — Banco Santander and Mastercard have executed the first regulated end-to-end payment by an AI agent in a live banking environment. Agentic payments are no longer a proof-of-concept — they are in production inside regulated institutions today.

- **[Oracle Expands AI Agent Studio for Fusion with Agentic Applications Builder](https://www.oracle.com/news/announcement/oracle-expands-ai-agent-studio-for-fusion-applications-with-agentic-applications-builder-2026-03-24/)** — March 24 announcement adds natural-language workflow composition, cross-agent orchestration, contextual memory, LLM multimodal support, and an ROI measurement dashboard to Fusion Applications. Non-developers can now compose multi-step agentic workflows — direct implications for FlexCube-adjacent back-office automation.

- **[LiteLLM Supply Chain Attack Hits 97M Downloads](https://www.theneuron.ai/ai-news-digests/around-the-horn-digest-everything-that-happened-in-ai-on-wednesday-march-25-2026-/)** — Malicious code was injected into LiteLLM, one of the most widely used LLM proxy/routing libraries. Any team using LiteLLM internally — common in enterprise AI stacks — should audit for affected versions immediately.

## Enterprise & Banking AI

- **[US Treasury Launches AI Innovation Series for Financial Sector](https://home.treasury.gov/news/press-releases/sb0421)** — Four-roundtable public-private initiative targeting AI operationalization in fraud detection, credit underwriting, and operational risk management. Treasury's framing is notable: AI in banking is now an "economic security" priority — regulatory tone is shifting from constraint to enablement.

- **[Spade Raises $40M Series B to Power AI-Driven Transaction Intelligence](https://fintech.global/2026/03/25/spade-raises-40m-series-b-to-power-ai-driven-finance/)** — Spade (backed by a16z, Oak HC/FT, and National Bank of Canada's VC arm) helps banks and fintechs make sense of high-volume, messy transaction data at scale. This is the enrichment layer that feeds fraud, reconciliation, and AML models — continued institutional capital flowing into AI payment infrastructure.

## Tools & Models

- **[Oracle AI Database 26ai Coming to Linux x86-64 On-Premises](https://blogs.oracle.com/database/oracle-ai-database-26ai-coming-soon-for-linux-x86-64-on-premises-platforms)** — Oracle's AI-native database edition is expanding from cloud to on-prem Linux deployments. For FlexCube environments with strict data sovereignty requirements, this opens a path to vector search and in-database ML without cloud dependency.

- **[Dapr Agents v1.0 Now Generally Available](https://dev.to/ajay_kumar_1daef5fe089885/ai-developer-tools-enter-autonomous-era-agentic-systems-rise-in-march-2026-1f38)** — CNCF's production-grade agentic orchestration framework ships v1.0 with enterprise reliability, security hardening, and durable workflows. The lowest-friction path to multi-agent systems for teams already running on Kubernetes or cloud-native infrastructure.

- **[Anthropic Launches Auto Mode for Claude Code](https://www.theneuron.ai/ai-news-digests/around-the-horn-digest-everything-that-happened-in-ai-on-wednesday-march-25-2026-/)** — Claude Code now executes multi-step tasks autonomously with fewer manual approvals; built-in safeguards review each action for destructive behavior before running. A meaningful productivity upgrade for development teams using AI-assisted engineering workflows.
